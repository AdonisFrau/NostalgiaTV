## NostalgiaTV 📺

**NostalgiaTV** is a personalized web-based environment designed to create a "vibe" for coding and studying[cite: 1]. It combines a retro-inspired aesthetic with modern development tools to provide a focused, atmospheric workspace[cite: 1].

---

### Core Concept
The project serves as a digital backdrop or "feed" that you can leave running in the background[cite: 1]. Whether it’s the dedicated **Nostalgiawire** feed or the main interactive app, the goal is to provide a consistent, nostalgic visual frequency while you work[cite: 1].

### Key Features
*   **Atmospheric Backgrounds**: The application features high-quality, immersive visuals, such as the full-screen cinematic backgrounds found in the `Nostalgiawire` feed[cite: 1].
*   **Minimalist Interface**: Designed to stay out of your way, the UI uses clean typography and subtle SVG iconography for social links (Discord, GitHub, Bluesky)[cite: 1].
*   **Modern Tech Stack**: Built with **Vite 8** and **TypeScript 6**, ensuring the environment is fast, lightweight, and easy to extend if you want to add more features[cite: 1].
*   **Tailwind CSS Styling**: Utilizes the latest **Tailwind 4** for styling, featuring a custom color palette optimized for both light and dark modes to reduce eye strain during long sessions[cite: 1].

### Project Components
| Component | Description |
| :--- | :--- |
| **The App** (`src/`) | The main interactive interface featuring a "hero" layout and a simple counter utility[cite: 1]. |
| **Nostalgiawire** (`feed/`) | A dedicated, distraction-free streaming feed page with a fixed cinematic background[cite: 1]. |
| **Assets** (`public/`) | Custom-designed lightning bolt favicons and branded SVG icons for a cohesive look[cite: 1]. |

---

### Quick Start for Your Sessions
To get your environment up and running for a study session:
1.  **Install dependencies**: `npm install`[cite: 1].
2.  **Launch the vibe**: `npm run dev`[cite: 1].
3.  **Build for hosting**: `npm run build` if you want to deploy it to a private URL to access from any device[cite: 1].
